# django-recipe-app-api

## Necessary Commands
```sh
docker build .
docker-compose build
docker-compose run --rm app sh -c "flake8"
docker-compose run --rm app sh -c "django-admin startproject app ."
docker-compose up
docker-compose down
docker-compose run --rm app sh -c "python manage.py startapp core"
```
